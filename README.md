# Luan & Cauane — Site de Casamento

Site do casamento construído com Vite, React, TypeScript e Tailwind CSS.

## Desenvolvimento

```sh
npm install
npm run dev
```

## Produção

```sh
npm run build
npm run preview
```

## Presentes

Todos os presentes são pagos via **PIX**. Ao clicar em "Presentear", o convidado vê a chave PIX e o valor.
